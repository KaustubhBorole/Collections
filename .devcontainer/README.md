# Devcontainer for Collections\n\nThis devcontainer makes the repository Spark/Codespaces-ready.\n\nWhat it does:\n- Uses the official TypeScript/Node devcontainer image.\n- Runs `npm ci` after creating the container to install dependencies.\n- Starts the Vite dev server on container start using `npm run dev -- --host` so the forwarded port is accessible.\n- Forwards port 5173 (Vite default) to make the app preview available in Codespaces/Spark.\n- Installs recommended VS Code extensions (Copilot Chat, Prettier, ESLint).\n\nHow to use:\n1. Create a Codespace from this repository (Code → Codespaces → New codespace) or open it in your GitHub Spark environment.\n2. The container build will run `npm ci` automatically.\n3. After the container starts, the dev server will automatically start and the forwarded port will show up in the Ports view; click "Open in Browser" to view the app.\n\nIf you prefer to start the server manually:\n- Remove or comment out `postStartCommand` in `.devcontainer/devcontainer.json` and run `npm run dev -- --host` in the terminal.
